##### Deploy to Heroku

- `sudo heroku login`
- `sudo heroku container:login`
- `sudo heroku create collection-kvn`
- `sudo heroku container:push web --app collection-kvn`
- `sudo heroku container:release web --app collection-kvn`
- `heroku open --app collection-kvn`
